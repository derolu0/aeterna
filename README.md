# Aeterna - Lexicon in Motu 🏛️

[English version below]

**Aeterna** è una *Progressive Web App (PWA)* di Informatica Umanistica dedicata all'analisi computazionale dei concetti filosofici. Il progetto applica metodologie di **Distant Reading** e **Digital Hermeneutics** per mappare l'evoluzione semantica del lessico filosofico attraverso i secoli.

## 🎯 Caratteristiche Scientifiche
* **Ermeneutica Algoritmica:** Motore di analisi rule-based progettato per l'identificazione di fratture concettuali e lo studio di termini derivanti da diverse tradizioni linguistiche (Greco, Latino, Italiano, Inglese, Francese, Tedesco).
* **Visualizzazione Topologica:** Generazione in tempo reale di grafi di conoscenza (Knowledge Graphs) tramite la libreria Vis.js per mappare le reti concettuali.
* **Geolocalizzazione del Pensiero:** Integrazione di mappe interattive (Leaflet.js) per contestualizzare geograficamente lo sviluppo delle scuole filosofiche.

## 🛠️ Architettura Tecnica
* **Frontend:** HTML5, CSS3, ES6+ JavaScript (Vanilla).
* **Data Strategy:** Struttura dati serverless basata su file JSON statici per garantire interoperabilità e sostenibilità digitale a lungo termine.

# Aeterna - Lexicon in Motu (English Version) 🏛️

**Aeterna** is a *Digital Humanities Progressive Web App (PWA)* dedicated to the computational analysis of philosophical concepts. The project implements **Distant Reading** and **Digital Hermeneutics** methodologies to map the semantic evolution of philosophical lexicons across centuries.

## 🎯 Scientific Features
* **Algorithmic Hermeneutics:** A rule-based engine designed to identify conceptual shifts and analyze terms originating from various linguistic traditions (Ancient Greek, Latin, Italian, English, French, German).
* **Topological Visualization:** Real-time Knowledge Graph generation using the Vis.js library to visualize conceptual networks.
* **Thought Geolocation:** Interactive maps (Leaflet.js) to provide geographical context to the development of philosophical schools.

## 🛠️ Technical Architecture
* **Frontend:** HTML5, CSS3, ES6+ JavaScript (Vanilla).
* **Data Strategy:** Serverless data structure based on static JSON files for long-term digital sustainability.
* **PWA Core:** Advanced Service Workers for an **Offline-First** experience, ensuring data access regardless of connectivity.
